OpenID Connect / OAuth2
=======================

The OpenID Connect community module (``gs-sec-oidc``) provides a single, unified OAuth2 / OpenID Connect
authentication connector for GeoServer. It supersedes the separate Spring Security 5 era plugins that were removed
in GeoServer 3.0:

* ``gs-sec-oauth2-openid-connect`` (also used to authenticate against **Azure AD / Microsoft Entra**)
* ``gs-sec-keycloak`` (authentication filter and role service)
* ``gs-sec-oauth2-google``
* ``gs-sec-oauth2-github``
* ``gs-sec-oauth2-geonode``
