Services
========

GeoServer serves data using standard protocols established by the `Open Geospatial Consortium <http://www.opengeospatial.org>`_.  These services are the primary way that GeoServer supplies geospatial information.

Supports requests for map images (and other formats) generated from geographical data.  

* :doc:`wms/index`

Supports requests for geographical feature data (with vector geometry and attributes).

* :doc:`wfs/index`

Requests for referenced raster data and imagery.

* :doc:`wcs/index`

Pre-rendered georeferenced map tiles generated from vector data, raster data and imagry.

* :doc:`wmts/index`

Analysis and transformation of geospatial vector data, raster data and imagery.

* :doc:`wps/index`

Discovery of geospatial resources using metadata records.
