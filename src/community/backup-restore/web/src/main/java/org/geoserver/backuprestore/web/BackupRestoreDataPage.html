Backup and Restore saves or reloads the GeoServer catalog and configuration (it does not copy the underlying data). Set the archive file path, optionally narrow the scope with the Workspace / Store / Layer filters, choose the options below (hover each option for guidance on what it does and when to use it), then start the job and watch its progress. It is good practice to run a Restore as a Dry-Run first.
