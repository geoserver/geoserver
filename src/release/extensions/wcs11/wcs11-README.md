GeoServer Web Coverage Service 1.1 Extension
============================================


About
-----

This extension provides support for the Web Coverage Service (WCS) 1.1.0 specification.
It allows clients to request coverage data (raster/gridded data) from a GeoServer instance.


Installation
------------

1. Extract the contents of the zip archive.

2. Copy the jar file(s) into the WEB-INF/lib/ directory in your GeoServer installation.
   This should include:
   - gs-wcs1_1-{version}.jar

3. Restart GeoServer by reloading the servlet or restarting your servlet container.


Documentation
-------------

For more information about WCS in GeoServer, see the GeoServer user manual:
https://docs.geoserver.org/latest/en/user/services/wcs/index.html
